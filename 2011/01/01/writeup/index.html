import rotor

def encrypt(data):
    key_a = '!@#$%^&*'
    key_b = 'abcdefgh'
    key_c = '<>{}:"'
    secret = key_a * 4 + '|' + (key_b + key_a + key_c) * 2 + '|' + key_b * 2 + 'EOF'
    rot = rotor.newrotor(secret)
    return rot.encrypt(data)


def decrypt(data):
    key_a = '!@#$%^&*'
    key_b = 'abcdefgh'
    key_c = '<>{}:"'
    secret = key_a * 4 + '|' + (key_b + key_a + key_c) * 2 + '|' + key_b * 2 + 'EOF'
    rot = rotor.newrotor(secret)
    return rot.decrypt(data)

intergrity

这个题目考的是AES-CBC
加密完成后的格式为 IV+AES-CBC(md5(pad(name))+pad(name))

这里因为只是验证了输入不为admin，
而name长度可以为32byte，所以直接可以构造name为md5(pad(name))+name，然后舍去原有IV向量，最后取后面的一部分，计算的时候就可以直接过。很棒。
