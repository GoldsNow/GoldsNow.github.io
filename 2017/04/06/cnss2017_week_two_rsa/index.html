cnss rsa

这个题目和BCTF的RSA200是一模一样的，但是搞懂rsa的同模攻击花了我好长的时间，这个真的需要有一定的数学基础才能够顺利的完成呢。

补充知识

参考了台湾大佬0xddaa的博客，了解一下模运算的几个重要的特性，才为最后能够解出题目做出铺垫：

  1. % 运算子优先度最后
  2. 满足足加法律
    • a + b % N = (a % N + b % N) % N
  3. 減法等同加上倒数, 因此也满足减法
    • a - b % N = (a % N - b % N) % N
  4. 乘法等于连加, 因此满足乘法
    • a * b % N = (a % N * b % N) % N
  5. 除法等同乘上倒数, 倒数就是模反元素
    • a * b_inv % N = (a % N / b % N) % N
  6. 指数等于连乘，因此满足指数律 (^ 表示平方)
    • (a * b) ^ r % N = (a ^ r % N) * (b ^ r % N) % N
    • (a - b) ^ r % N = (a ^ r % N) / (b ^ r % N) % N
    • g ^ (a + b) % N = (g ^ a % N) * (g ^ b % N) % N
  7. 任何数乘上模反元素的余数会是 1
    • a * a_inv % N = 1
